PHP com MVC

Model

Gerencia dados e lógica de negócios. O modelo define quais dados o aplicativo deve conter.

View

Lida com layout e exibição. Define como os dados do aplicativo devem ser exibidos.

Controller

Define como os dados do aplicativo devem ser exibidos. O controlador contém lógica que atualiza o modelo e/ou visualização em resposta à entrada dos usuários do aplicativo.
